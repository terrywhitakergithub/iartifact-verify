# iartifact-verify
Client-side verifier for iArtifact certificates — runs entirely in your browser, nothing uploaded.
Public verifier for iArtifact certificates.

Drop an artifact folder to independently verify:
- Every file hash matches what was recorded at creation
- The provenance chain is intact and unbroken  
- The RFC 3161 timestamp is valid
- The seal is present and unaltered

Nothing leaves your machine. No server. No account. Open source.
